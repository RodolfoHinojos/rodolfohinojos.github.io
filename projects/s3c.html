S3C – Sistema de Control de Crédito a Cliente

Sistema interno desarrollado para tienda departamental (lote de autos y mueblería), integrado con el ERP Microsip.

Contexto

La empresa requería garantizar que únicamente clientes con información completa y validada pudieran ser habilitados para financiamiento a plazos dentro del ERP.

Arquitectura Técnica

Base de Datos

  • Motor: Firebird
  • Base independiente para información extendida del cliente
  • Conexión directa a base de datos de Microsip

Modelo de Integración

  • Lectura y actualización directa en base de datos del ERP
  • Uso de transacciones explícitas
  • Rollback ante errores

Lógica de Negocio

  • Validación de campos obligatorios
  • Detección de clientes duplicados
  • Control manual de habilitación
  • Separación entre evaluación financiera y operación comercial

Modelo Multiusuario

  • Arquitectura cliente-servidor
  • Roles: Administrador / Capturista
  • Operación concurrente en dos sucursales

Impacto

  • Reducción de clientes duplicados
  • Mayor control en habilitación de crédito
  • Estandarización de captura de datos
  • Disminución de errores operativos

Este proyecto demuestra experiencia en integración ERP, control de integridad de datos y manejo de transacciones en entornos empresariales.
